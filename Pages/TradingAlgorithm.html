AI Stock Trading Project (Work in Progress)

An AI-driven trading algorithm designed to analyse market data and execute trades.









































About The Project

This project is an algorithmic stock trading algorithm developed as a personal endeavour to explore algorithmic trading. The system is built upon the Pybroker framework and is written in Python. This program also integrates into the Alpaca API to fetch market data and also will be able to execute trades. Use of the Pybroker framework allows for machine learning and data analysis to be implemented into the program. This project is still very much a work in progress and is still under development.

Features

Market Analysis

Has the ability to utilise algorithms and indicators to analyse market trends in order to execute trades.

Automated Trading (Planned)

Executes trades automatically based on predefined strategies and market conditions.

Customizable Strategies

Allows users to customize trading strategies based on their risk tolerance and investment goals.

Data Integration

Integrates with various APIs to fetch real-time market data and historical prices.

Backtesting

Utilises the power of Pybroker to backtest an algorithm with historical data to evaluate its performance.

Machine Learning

Utilising Pybroker's backtesting and "walkforward analysis" to train a model on historic data to evaluate its performance on data it has not yet seen.

Visualisation

Uses Matplotlib to visualise the performance of the algorithm and a portfolio over time in backtesting.

Metrics

Bootstrap metrics offer a more reliable method for evaluating algorithm performance during backtesting by using randomly sampled subsets of the dataset

Review

What I've Learned So Far

This project has given me the opportunity to improve my python programming skills, a language I hadn't touched since GCSE, and learn how to work with API's, libraries and frameworks. It has also given me a better understanding of algorithmic trading and has taught me some of the basics of machine learning.

Challenges Faced

Learning how to use frameworks has been a challenge, especially with PyBroker. Not only is it new to me, but it's also written in a language I haven't used in a while. On top of that, the entire premise of algorithmic trading is inherently complex, involving advanced mathematics and statistics.

Future Improvements

I am currently undergoing a rewrite of what I have done so far to use more Object-Oriented Programming principles, improving its robustness, making it easier to read, as well as increasing scalability.


After the rewrite, I plan to work on creating more advanced machine learning features as the current algorithm, which is built upon a simple linear regression model, is limited. After creating a profitable model and strategy in backtesting scenarios, I then hope to add functionality for the program to be able to live trade by making use of the Alpaca API.
